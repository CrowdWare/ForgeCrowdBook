# i18n

## Goal
Load translation strings from `i18n/strings-{lang}.sml` files at startup and provide a lookup function.

## Context
Uses `sml-go` to parse SML files. No external i18n library.
Supported languages: de, en, eo, pt, fr, es.

## Tasks

### 1 — Create `i18n/strings-en.sml` (and all 5 other languages)

### 2 — `internal/i18n/i18n.go`
```go
type Bundle struct {
    strings map[string]map[string]string // lang → key → value
}

func Load(dir string) (*Bundle, error)
func (b *Bundle) T(lang, key string) string
func (b *Bundle) Map(lang string) map[string]string
```
- `Load` reads all `strings-{lang}.sml` files in `dir`.
- `T` returns the value for `key` in `lang`, falls back to `en`, then returns `key` if missing.
- `Map` returns the full map for a language (passed to templates).

### 3 — Language detection helper
```go
func DetectLang(r *http.Request, supported []string) string
```
- Checks `lang` cookie first.
- Falls back to `"en"`.

## Acceptance Criteria
- All 6 language files parse without error.
- `T("de", "nav_home")` returns `"Startseite"`.
- Unknown key returns the key itself (never panics).
- Missing language file logs a warning and continues.
